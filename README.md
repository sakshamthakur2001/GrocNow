# 🛒 GrocNow - Online Grocery App (UI/UX Design)

## 📖 Project Overview

GrocNow is a modern and user-friendly Online Grocery Shopping App designed to provide customers with a seamless digital shopping experience. The project focuses on creating an intuitive and visually appealing user interface using **Figma**, following modern UI/UX design principles.

The application allows users to browse grocery products, search items, manage their shopping cart, place orders, track deliveries, and manage their accounts with a simple and efficient user experience.

---

## 🎯 Project Objectives

- Design a clean and modern grocery shopping interface.
- Improve the overall user experience.
- Simplify online grocery shopping.
- Create an intuitive navigation system.
- Build a complete shopping workflow from Login to Order Tracking.
